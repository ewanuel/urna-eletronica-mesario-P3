Botões Gerais

O botões que aparecem em todas as páginas:

  • Modo Escuro/Claro Alterna entre o tema escuro e claro do sistema
  • Menu burger: Um menu com atalhos para os principais locais do sistema
  • Voltar: volta para a página anterior
  • Duvida: Apresenta a função de todos os mecanismos da página


Menu principal

O Menu principal apresenta as opções essenciais para o processo eleitoral tais como:

  • Gerenciar Candidatos: Cadastro e visualização dos grêmios ou presidenciaiveis
  • Gerenciar Eleitores: Cadastro, visualização e edição dos eleitores
  • Administração: Execução, edição e visualização de informações sensíveis, só é vísivel para usuários administradores
  • Iniciar Processo: Executa o processo de votação autorizando eleitores ao voto


    • Menu de Candidatos

    O Menu de gerenciamento dos candidatos apresenta ferramentas para administrar o fluxo de grêmios e presidenciáveis tais como:

    • Cadastrar nova chapa: Inseri um novo candidato para a eleição
    • Exibir todos os candidatos: Lista todos os candidatos existentes com a possiblidade de edição e remoção



      • Cadastro de Chapas

      O Cadastro de Candidatos possíbilita a inserção de candatos ou grêmios para a eleição

      • Digite o número: Inserção do número do presidenciável que será utilizado no processo eleitoral
      • Digite o nome: Inserção do nome do candidato que será exibido ao eleitor após introdução do número no processo eleitoral
      • Foto do Candidato: Inserção da imagem que representa o presidenciável e que será exibida no processo eleitoral
      • Cadastrar: Inseri o presidenciável para concorrer as eleições



        • Gerenciador de Chapas

        O Gerenciador de Candidatos possíbilita a visualização e edição desses objetos

        • Digite o Nome: Pesquisa e exibe o candidato por meio do nome
        • Deletar Candato: O ícone de lixeira deleta o candidato individualmente:
        • Editar Candidato: O ícone do lapís permite alterações no candato:
        • Deletar Todos: Deleta todos os candatos existentes


          • Menu de Eleitores

          O Menu de gerenciamento dos eletores apresenta ferramentas para administrar o eleitorado tais como:

          • Adicionar eleitor: Inseri um novo eleitor
          • Exibir todos os eleitores: Lista todos os eleitores existentes com a possiblidade de edição e remoção



            • Cadastro de Eleitores

            O Cadastro de Eleitores possíbilita a inserção dos candidatos para votar na eleição

            • Classe: Inserção do número e letra correspondente a classe dos eleitores
            • Números (um por linha): Cada linha do campo representa o numero de um aluno na chamada
            • Nomes (um por linha): Cada linha do campo representa o nome de um aluno
            • Cadastrar: Cria um RA para a lista de alunos possibilitando e credenciando de um voto por pessoa



              • Gerenciador de Eleitores

              O Gerenciador de Eleitores possíbilita a visualização e edição desses objetos

              • Digite o Nome: Pesquisa e exibe o eleitor por meio do nome
              • Selecionar eleitor: O ícone de seleção armaza o item para uma possível remoção posterior:
              • Deletar eleitor: O ícone de lixeira deleta o eleitor individualmente:
              • Editar eleitor: O ícone do lapís permite alterações no eleitor:
              • Excluir Selecionados: Remove apenas os eleitores marcados pelo icone de seleção:
              • Deletar Todos: Deleta todos os eleitores existentes



                • Menu Administrativo

                O Menu do administrador apresenta ferramentas para edição e visualização de dados sensíveis tais como:

                • Apuração dos votos: Mostra o resultado do processo eleitoral
                • Alteração de senhas: Altera a senha dos mesários
                • Apagar todos os dados: Apaga todos os votos, candidatos e eleitores voltando o sistema em seu estado original



                  • Apuração dos votos

                  A contagem de votos exibe o resultado eleitoral

                  • Encerrar Processo: Apaga todos os votos



                    • Alteração de senhas

                  • Selecione o tipo de usuário: Seleciona o mesario ou administrador para alteração de senha
                  • Digite a nova senha: Campo de inserção de senha
                  • Repita a nova senha: Campo para validar a senha
                  • Salvar: Efetua a alteração


                    • Liberação de votos

                    O Menu de liberação permite que os eleitores exerçam seu direito ao voto

                    • Campo de busca: Permite pesquisar o eleitor por RA, Nome ou Classe
                    • Liberar Voto: Permite que o eleitor vote, caso o mesmo já tenha votado aparecera a mensagem:"Eleitor já votou!"
